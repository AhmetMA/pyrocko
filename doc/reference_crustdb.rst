The :mod:`crustdb` Module
==========================

Select and statistically analyse empiracal velocity profiles gathered in the `Global Crustal Database <https://earthquake.usgs.gov/data/crust/>`_.

.. note ::

    **Citation:**

    W.D. Mooney, G. Laske and G. Masters, CRUST 5.1: A global crustal model
    at 5°x5°. J. Geophys. Res., 103, 727-747, 1998.
